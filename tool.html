What is Overfitting in supervised learning?

Let’s find out!

If you are an AI developer, you must have wondered that what is Overfitting exactly?

So, every neural network has three types of layers: input layer, hidden layer and output layer. And therefore, creating a NN means coming up with an appropriate number of layers and a number of nodes in each one of these layers.

Input and output layers:
These two layers are always going to be single layers in almost all types of NN architectures. The number of nodes in each layer will also be one single node.

The Hidden Layers: In hidden layers, the case is different.
If your data is linearly separable, then you do not need any hidden layer. However, Although a single hidden layer is sufficient for the large majority of problems, sometimes there could be a huge improvement by adding 2nd or 3rd layer.


Overfitting issue: Overfitting happens when your model learns the noise along with the data during training it and will definitely negatively impact the performance of it. This means that the noise will be taken with the data by the model and will reduce the ability of the model to generalize.
